# <a name="author-not-found"></a>author-not-found

## <a name="warning"></a>警告

`Invalid value for author: '{value}' is not a valid GitHub ID.`

## <a name="resolution"></a>解決方式

新增有效的 GitHub 識別碼作為 `author` 值。
